login  [line 38]

integer login( $args  )

Allows a registered user to login into the board.

You need to provide 4 parameters inside the $args array. At this time we don't support new user registration directly via API.
To create a new account you need to browse the board using a browser.

PARAMETERS:

  • $username [string] registered username
  • $password [string] username associated password (in clean format)
  • $autologin [boolean] auto login the next time?
  • $viewonline [boolean] show user status in forum statistics?

 Parameters:
   $args: 

API Tags:
Return:  LOGIN_SUCCESS (login succeded)
LOGIN_ERROR_PASSWORD (wrong given password),
LOGIN_ERROR_ATTEMPTS (you have reached the maximum login attempts, your account was blocked)
See:  FUNCTION DOES NOT REQUIRE LOGIN (uh uh, that's a login function baby!)

loginProtectedForumWithPassword  [line 87]

integer loginProtectedForumWithPassword( $forum_id, $password  )

This function is used to login in a password-protected forum.

You should not use it directly but it's called when you use getTopicsFromForum function (to return the list of topics into the forum) when you give a $forumPassword.

PARAMETERS:

  • $forum_id [string] the id of the forum you want to login
  • $password [string] given password to access

 Parameters:
   $forum_id: 
   $password: 

API Tags:
Return:  LOGIN_FPSWD_OK you are successfully logged into the forum
LOGIN_FPSWD_ERR given password is wrong
LOGIN_PSWD_FORUM_ALREADYLOGGED you are already logged into the forum
LOGIN_FPSWD_WRONGFORUMID given forum id does not exist
