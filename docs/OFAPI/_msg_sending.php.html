Delete post from a forum


PARAMETERS:

  • $forum_id [integer] specify forum id
  • $topic_id [integer] specify topic id
  • $post_id [integer] specify post id
  • $forumPassword [integer] specify password if needed

 Parameters:
   $args: 

API Tags:
Return:  AVAILABLE ERRORS MESSAGES:

    • MSG_FORUM_ERR_FORUMPASSWORDED forum is password protected and you have not specified a key
    • BOT_GO_HOME bot cannot act in this way (you have a bot?)
    • USER_CANNOT_READ_FORUM user cannot read specified forum
    • USER_CANNOT_POST user cannot post to this forum
    • FORUM_LOCKED specified forum is locked
    • NOT_AUTHORISED you're not authorized

    WHEN MESSAGE IS POSTED THIS METHOD AN ARRAY WITH THESE KEYS:
